Quick Rules

  • 5x5 grid, center is locked.
  • Jerry = score. Every 5 jerry gives +1 reserve.

  • Water Girl (odd spawns): instant win.
  • Nuke (even spawns): instant loss.
  • Dead Fish: -1 main, -1 reserve.
  • Yellow Skull: -1 main, +1% Nuke chance.
  • Bio: -1 main, good token duration drops (10 clicks).
  • Well: +1 main or reserve, +1% Water Girl chance.
  • Spring: +1 main, next good tokens bounce once (10 clicks).
  • Pool: +1 main, +1 reserve.
  • Recycling: -1 extra click, +1 main score.
  • Gas Mask: +1 click, -1 reserve score.
  • Every 7 reserve: removed for +3 main score.
